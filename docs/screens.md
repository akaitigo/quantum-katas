# 画面設計

## 画面遷移図

```mermaid
stateDiagram-v2
    [*] --> KataList: アクセス
    KataList --> KataDetail: カタ選択
    KataDetail --> KataDetail: コード編集・実行
    KataDetail --> KataDetail: ヒント表示
    KataDetail --> KataList: 戻る
    KataDetail --> KataDetail: 正解 → 次のカタへ
```

## レスポンシブ対応

| 画面幅 | レイアウト |
|--------|-----------|
| >= 1024px | 2カラム (解説 | エディタ+結果) |
| 768-1023px | 2カラム (狭め) |
| < 768px | 1カラム (解説 → エディタ → 結果 縦積み) |
